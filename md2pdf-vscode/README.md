# md2pdf — Markdown を PDF に変換（VSCode / Cursor 拡張）

`.md` を右クリックして PDF に変換する拡張機能です。用途に応じて 2 つの出力に対応します。

- **PDFに変換(1ページ)** … 末尾余白のない**連続1ページ**PDF（ページ区切りなし）
- **PDFに変換(改ページあり)** … **A4縦**で出力。章（`##` 見出し）を新しいページから開始しつつ、章内は詰めて配置

画像・表・コードブロックに対応します。

## 使い方

- 右クリック（エクスプローラ / エディタ / エディタタブ）→ **「md2pdf」** サブメニュー
  - `PDFに変換(改ページあり)`
  - `PDFに変換(1ページ)`
  - `HTML(検証用)を出力` … レンダリング後のHTMLを `<名前>.html` として保存（ブラウザで開いてレイアウト確認用）
  - `デフォルトCSSを出力` … 同梱スタイルを `md2pdf-default.css` として出力（編集して `cssPath` に指定すれば差し替え可能）
- コマンドパレットからも実行可能（`md2pdf: …`）
- 生成後に「開く / フォルダで表示」を選べます

### レイアウトの特徴（改ページあり）

- 章（`##`）は必ず新しいページから開始（ただし先頭の見出し＝目次などはタイトルと同じページに残す）
- 節（見出し＋本文）はページ内で分断せず、収まらない場合は丸ごと次ページへ
- 「説明文＋直下の画像」は分断せず同じページに保つ（収まらなければ文章ごと次ページへ）
- 目次などの**内部リンク**や外部 URL がクリック可能

## 動作要件（ブラウザ）

変換には Chrome 系ブラウザが必要です。次の優先順で**既存のブラウザを自動検出**して使います。

1. 環境変数 `PUPPETEER_EXECUTABLE_PATH`
2. Puppeteer が取得済みの Chromium（共有キャッシュ）
3. システムにインストール済みの **Google Chrome / Microsoft Edge / Chromium**

そのため、Chrome か Edge が入っていれば追加のダウンロードは不要です。
いずれも無い場合は、一度だけ `npx puppeteer browsers install chrome` を実行してください（変換失敗時のエラーメッセージにも同じ手順を案内します）。

## 設定（settings.json）

| 設定キー | 既定 | 説明 |
|---|---|---|
| `md2pdfOnepage.width` | `210` | ページ幅（mm）。「1ページ」モードのみ有効（改ページありは A4 固定） |
| `md2pdfOnepage.margin` | `15` | 余白（mm） |
| `md2pdfOnepage.cssPath` | `""` | 適用CSSの絶対パス（空なら同梱スタイル） |
| `md2pdfOnepage.openAfter` | `true` | 生成後にPDFを開く |
